# Housing TFM Thesis

Research question:
Do tabular foundation models improve the prediction of cross-sectional housing prices relative to classical hedonic regression models and traditional machine-learning methods, and what trade-offs arise in interpretability, robustness, and predictive uncertainty?

Datasets:
- Ames Housing
- California Housing
- King County House Sales

Planned models:
- Hedonic OLS
- Log-linear hedonic regression
- Random forest
- Gradient boosting / XGBoost
- TabPFN
